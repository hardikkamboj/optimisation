About

The conic hull (or conic combination) of a set of points is the set of all non-negative linear combinations:


cone(S) = { θ₁x₁ + θ₂x₂ + ... | θᵢ ≥ 0 }


Unlike the convex hull, coefficients don't need to sum to 1 — they just need to be non-negative. The result is a cone with apex at the origin.
