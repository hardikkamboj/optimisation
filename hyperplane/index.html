About

A hyperplane is the set { x | aᵀx = b }. In 2D this is a line.


A hyperplane divides space into two halfspaces:


{ x | aᵀx ≥ b } and { x | aᵀx ≤ b }


The normal vector a is perpendicular to the hyperplane and points toward the aᵀx ≥ b side.
