Visualizing Optimization

Interactive visualizations of fundamental optimization concepts

Convex Hull

Click to place points on a 2D canvas and compute the convex hull — the smallest convex polygon enclosing all points. Uses the Graham Scan algorithm.

Conic Hull

Visualize the conic hull — the set of all non-negative linear combinations of points. See how rays from the origin form a cone that contains all the points.

Hyperplanes & Halfspaces

Visualize the hyperplane aᵀx = b and the two halfspaces it defines. Adjust the normal vector and offset to see how the hyperplane moves.

Polyhedra

Build a polyhedron by adding linear constraints (halfspaces). See how the feasible region P = { x | Ax ≤ b } changes as you add or remove constraints.

Linear Programming

Visualize a linear program: minimize cᵀx over a feasible polyhedron Ax ≤ b. See how level curves of the objective sweep across the feasible region and where the optimum lands.

Quadratic Programming

Visualize a quadratic objective f(x) = xᵀQx + cᵀx. See how the shape of the level curves changes with Q — ellipses when Q is positive definite, hyperbolas when indefinite.
